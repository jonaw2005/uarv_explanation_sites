⚠️ Important Wiring Notes:
• Power: Always connect the Power Module before connecting USB. The Pixhawk can be powered via USB for configuration, but the power module provides the main flight power.
• Motor Order (Quadcopter X-Frame): MAIN OUT 1 = Front-Right (CW), MAIN OUT 2 = Rear-Left (CW), MAIN OUT 3 = Front-Left (CCW), MAIN OUT 4 = Rear-Right (CCW). Verify with your frame manual.
• ESC Calibration: If using PWM ESCs, calibrate them through the Pixhawk before first flight. DShot ESCs do not require calibration.
• Telemetry: TELEM1 is typically used for the primary telemetry radio. TELEM2 can be used for a secondary radio or connected to an onboard computer (Raspberry Pi / Jetson).
• GPS: Mount the GPS/Compass module on a mast away from magnetic interference (ESCs, high-current wires). The compass direction arrow should point toward the front of the vehicle.
• RC Receiver: For SBUS receivers, connect to the RC IN port. For PPM receivers, connect to the same port. For Spektrum satellite receivers, use a special adapter or connect to a UART.
• Safety: The safety switch must be pressed (LED turns solid) before arming. The buzzer provides audible arming/disarming status.
• Firmware: Install ArduPilot (ArduCopter / ArduPlane) or PX4 Pro via QGroundControl or Mission Planner.
